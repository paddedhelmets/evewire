# evewire

Modern EVE Online pilot and altcorp management tool.

A from-scratch rebuild inspired by [evething](https://github.com/madcowfred/evething), 
using the modern ESI (EVE Swagger Interface) APIs.

## Status

Discovery phase - analyzing evething and designing architecture.

## Goals

- Blazing fast performance
- Modern ESI API integration (OAuth2)
- Clean, maintainable codebase
- Focus on pilot/altcorp management (not alliance-scale)
